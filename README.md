## About

**Sea Battle Game** is a classic "Battleship" game written in C++.
Two players place ships on 10×10 boards and fire at the opponent’s coordinates. The winner is the player who destroys all enemy ships first.

Features:

- Interactive console interface
- Undo the last placed ship and reset all placed ships

## Documentation

- Board size: 10×10
- Ships: lengths 4, 3, 2, 1 (multiple copies)
- Ship placement: enter `x y` and orientation (`H` horizontal, `V` vertical, default is `H`)
- Undo last ship: `U`
- Reset all ships: `R`
- Shoot: enter `x y` to attack the opponent’s board
- Players take turns until all ships of one player are destroyed

## About

**Sea Battle Game** — это классическая игра "Морской бой" на C++.
Два игрока размещают корабли на досках 10×10 и стреляют по координатам противника. Побеждает тот, кто первым уничтожит все корабли соперника.

Особенности:

- Интерактивный консольный интерфейс
- Отмена установки корабля и сброс всех установленных кораблей

## Documentation

- Размер доски: 10×10
- Корабли: длины 4, 3, 2, 1 (несколько копий)
- Размещение кораблей: ввод `x y` и ориентация (`H` горизонтально, `V` вертикально, по умолчанию `H`)
- Отмена последнего корабля: `U`
- Сброс всех кораблей: `R`
- Выстрел: ввод `x y` для атаки по доске противника
- Игроки ходят по очереди до уничтожения всех кораблей одного из игроков
